# Excel Multi-Partner Reporting Automation

## Overview
This project demonstrates an end-to-end Excel-based reporting automation system for consolidating daily transaction files from multiple partners.

## Architecture
Daily Transaction Files (8 Partners)
→ Power Query Standardization
→ Monthly Partner Dataset
→ Master Consolidated Dataset
→ VBA Orchestrator with Run Logging

## Key Features
- Multi-partner daily data ingestion
- Header standardization using Power Query
- Monthly consolidation per partner
- Master dataset aggregation
- One-click refresh using VBA macro
- Automated run logging (status, duration, errors)

## How to Run
1. Open `automation refresh.xlsm`
2. Enable macro
3. Click Refresh
4. Check log sheet
5. Review consolidated output

## Tools Used
- Microsoft Excel
- Power Query
- VBA Macro
